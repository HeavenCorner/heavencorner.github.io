HTML5标签-03表单

03 表单元素

input新类型

tel

用于输入电话号码，用户点击识别后会跳出数字键。

<input type="tel"/>

显示效果

search

用于提示用户输入要搜索的文字；search和text之间的区别主要在于样式上

<input type="search"/>

显示效果

url

用于输入单个网址

<input type="url"/>

显示效果

email

用于输入单个电子邮件地址；如果指定了multiple属性，就可以输入多个电子邮件地址，以逗号分隔

<input type="email"/>

显示效果

datetime

用于输入时区设置为 UTC 的日期和时间。

<input type="detetime">

####显示效果

date

用于输入不含时区的日期

<input type="date">

####显示效果

month

用于输入含年份和月份但不含时区的日期

<input type="month">

####显示效果

week

用于输入含年份和周数但不含时区的日期；例如：此格式的日期2011-W05表示2011年的第5周。

<input type="week">

####显示效果

time

用于输入含小时、分钟、秒和秒的小数部分但不含时区的时间值。

<input type="time">

####显示效果

datetime-local

用于输入不含时区的日期和时间

<input type="detetime-local">

####显示效果

number

用户输入数字

<input type="number">

####显示效果

range

用于输入数字，但与number的区别在于无需输入具体数字；在大部分支持该类型的浏览器中，范围控件的实施形式为滑块。

<input type="range">

####显示效果

color

用于通过颜色池控件选择颜色（有兼容问题）

<input type="corder">

####显示效果

其他新增类型

lable

表单关联元素，通过增加id选择器和for关联，点击关联值即可聚焦输入框

<lable for="zh">账号</lable>

<input type="text" id="zh"/>

显示效果

    账号

datalist

定义选项列表，配合input使用，来定义input可能的值；id选择器定位，list关联id名。

    <input type="text" list="haha"/>
    <datalist id="haha">
        <option value="香蕉"></option>
        <option value="苹果"></option>
        <option value="哈密瓜"></option>
        <option value="啤酒"></option>
        <option value="饮料"></option>
    </datalist>

显示效果

keygen

元素的作用是提供一种验证用户的可靠方法。

元素是密钥对生成器（key-pair generator）。当提交表单时，会生成两个键，一个是私钥，一个公钥。

    <form action="demo_form.asp" method="get">
    Username: <input type="text" name="usr_name" />
    Encryption: <keygen name="security" />
    <input type="submit" />
    </form>

显示效果

    

    Username: 
    Encryption:

ins和del

定义已经被插入文档中的文本

<ins>1111</ins> <del>11111</del>

显示效果

1111 11111

samp

表示一段用户应该对其没有什么其他解释的文本字符。要从正常的上下文抽取这些字符时，通常要用到这个标签。

字符序列 <samp>ae</samp> 可能会被转换为 &aelig; 连字字符。

显示效果

    字符序列 ae 可能会被转换为 æ 连字字符。

fieldset

元素可将表单内的相关元素分组，legend用来定义fieldset的标题。

<fieldset>
    <legend>用户信息</legend>
    <label for="username">帐号</label>
    <input type="text" id="username"/><br/>
    <label for="pas">密码
    <input type="password" id="pas"/><br/>
</fieldset>

显示效果


        用户信息

legend

用来定义fieldset的标题，见上。

表单属性

autofocus

用于在网页加载后对焦到相关元素上的输入。

autofocus 的目标可以是输入、选择、文本区域和按钮。

<input type="text" autofocus/>

显示效果(自动选中，，当你打开这篇文章的时候，它自动跳到了小面的输入框)

placeholder

用于提示用户应输入的数据类型。

在对焦到相关元素以及用户输入数据之前，系统会以浅色文字显示占位符值。您可以在输入和文本区域中指定该值。

<input type="text" placeholder="请输入你的帐号"/>

显示效果

required

规定必需在提交之前填写输入字段

<input type="text" name="usr_name" required="required" />

<input type="submit" />

显示效果

pattern

规定用于验证输入字段的模式

国家代码：<input type="text" name="country_code" pattern="[A-z]{3}" title="三个字母的国家代码" />

<input type="submit" />

显示效果

国家代码：

autocomplete

规定表单是否应该启用自动完成功能

<form action="/example/html5/demo_form.asp" method="get" autocomplete="on">

First name:<input type="text" name="fname" /><br />

Last name: <input type="text" name="lname" /><br />

E-mail: <input type="email" name="email" autocomplete="off" /><br />

<input type="submit" />

</form>
