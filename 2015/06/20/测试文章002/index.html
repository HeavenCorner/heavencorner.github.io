测试文章002

一、前期准备

本篇教程基于Mac快速搭建个人博客

1、安装NodeJS

  1. 可以通过官网下载NodeJS。
  2. 运行安装nodeJS。
  3. 安装完成后在命令行输入“node -v”即可判断是否安装成功

显示为v+数字组合就是安装的node的当前版本，表示已经安装成功

2、安装Git

  1. Mac可以通过安装Xcode就自带有Git
  2. 在命令行输入“git –version”可查看Git是否安装成功

显示为git version+数字组合就是git的当前版本号，表示已经安装了git

3、申请github账号

点击进入github官网申请账号

二、Hexo的安装

  1. 首先在命令行里输入命令“hexo install -g hexo”
  2. 如果安装过程中有出现报错字样的话，可以尝试下用sudo来提升安装权限，输入命令：“sudo hexo install -g hexo”(不行就“sudo npm install-g hexo”)。
  3. 然后会提示输入密码，即为电脑管理员的密码。
  4. 在命令行里输入“hexo version”，看到hexo的相关信息即表明hexo安装成功。
  5. 新建一个文件夹，用于存放整个博客，在此文件夹下输入命令“hexo init”初始化hexo目录。
  6. 最后输入“hexo server”启动本地配置，显示如下：

    INFO  Start processing
    INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.
  7. 复制上面代码中的地址到浏览器便可以在本地浏览新建的博客。

三、配置Github

已申请的Github账号一枚

1、配置ssh-key

  1. 在命令行里输入“git”，等待运行。
  2. 继续输入“ssh-keygen”然后连续三次回车，可以看到下面语句
    The key’s randomart image is:
    +—[RSA 2048]—-+
    |             o.++|
    |           . .+o|
    |          = +++ B|
    |         B =.o+B.|
    |        S      +.o=+|
    |         = E ooo=|
    |          = +  oo|
    |           o +. .|
    |            +.o+.|
    +—-[SHA256]—–+
  3. 接着输入“vim ~/.ssh/id_rsa.pub”会出现另一个对话框，复制“ssh-key”（注意不要多复制内容，否则无效），然后输入“:q”退出窗口，回到命令行。
  4. 登录Github，进入设置-SSH and GPG keys-New SSH key把复制的ssh-key添加进去。

2、配置Github与本地机器验证

  1. 在命令行中输入“git config - -global user.name “此处为申请的Github的用户名””
  2. 继续输入“git config - -global user.email “此处为申请的Github邮箱””

3、新建博客域名仓库

  1. 点击“new repository”新建域名仓库，编写输入域名名称。
  2. 勾选“ Initialize this repository with a README ”选项，提交。

四、发布

1、发布前的准备

  1. 打开新建的博客文件夹，根目录下找到并使用“Sublime Text”打开“_config.yml”文件.
  2. 文档开头的语句可以自定义，关于博客的名称和相关简介标签。
    title: 博客标题名称
    subtitle: 副标题
    description:
    author: 博主
    language:
    timezone:
  3. 在文档的最下边，修改“type: git”和添加“repo: 域名仓库的https地址”，代码如下：

    deploy:
    type: git
    repo: https://github.com/-----------/-----------.io.git

2、正式发布

  1. 部署提示找不到Git,输入命令“npm install hexo-deployer-git –save”等待安装完成。

    npm install hexo-deployer-git –save
  2. 输入命令“hexo g”转换文本格式。
  3. 输入命令“hexo d”发布上传到服务器库。
  4. 可以了，登录浏览器查看你的微博吧~
