一 • 幻夜

去年的今天，躺在松江区一间宾馆的床上，盯着天花板发呆，突然有点渴了，便随意收拾了一下，看了手机21点多，我跪下祷告之后，准备到附近的超市买水。

路上在想怎么度过这个夜晚，晚上厂区附近的那条街道人还是挺多的，就像之前学校的后街一样，无非是到处乱走的学生换成了到处晃悠的职工；看到一家比较大的超市，就走了进去，拿了一瓶水，在我选牙膏的时候，身边的超市人员给我介绍另一款牙膏，我笑了一下没理她，就去了收银台。

从超市出来，有种很厌倦的感觉，反感到周围每一个经过的人，直到那些矛头最后都又指向了自己。于是我抬头看了看天空，这个夜晚，这个地点，一切都已经在这里终结，暂时无法承受发生的事，于是缓解悲伤而暂时寄托于幻想中，只能延期处理，的确只有夜晚这个场景比较适合，尽管不知道这种因为悲伤事件而产生的幻想保护层会维持多久。

另一个旁观者的自己，当初是怎么警告我的，现在说已经没有意义了，我的路好像已经走到了尽头，先不要取代我，给我一个缓冲期，之后，就躲在那个死胡同里，不再掺和所有的一切，那里没有光，本来就是我该去的地方，就像现在的夜晚一样。只有不断的实体成就才能堵住那扇门，不让我靠近，这样就不会搞砸一切了。而你，除了可以堵住那扇门，还曾一次一次实现我幻想的一切。

事实是，我把事情搞砸了，要快点离开这个仅呆了不到一天的环境，电话里的嘲笑，当事者的讽刺，环境、同事价值观的影响，尤其是此时无能为力的自己。没有工作、没有存款、没有技能、没有勇气，没有交际、没有价值，什么也没有。开始憎恨身边所有的一切，这种怨恨渐渐超出了自身承受能力。此时支撑我的，即便失去一切，还有信仰，那是来自灵魂深处的安全感，我会更强，更好。
